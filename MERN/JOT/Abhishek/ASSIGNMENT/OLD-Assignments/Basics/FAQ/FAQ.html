Frequently Asked Questions


Question 1: What is HTML?

Answer: HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It defines the structure and content of web pages.


Question 2: How do I create a hyperlink in HTML?

Answer: To create a hyperlink, use the anchor tag <a>. For example: <a href="https://example.com">Visit Example</a>


Question 3: What is the difference between HTML and XHTML?

Answer: HTML is more lenient with syntax, while XHTML is stricter and requires well-formed XML. XHTML also requires lowercase tags and attributes.


Question 4: How do I create a table in HTML?

Answer: Use the <table> element along with <tr> for rows, <th> for headers, and <td> for data cells.


Question 5: What is the purpose of semantic HTML?

Answer: Semantic HTML uses elements that clearly describe their meaning to both the browser and the developer. This improves accessibility, SEO, and code maintainability.


Question 6: How do I add an image to an HTML page?

Answer: Use the <img> tag with the src attribute for the image URL and alt for alternative text. Example: <img src="image.jpg" alt="Description">


Question 7: What is the use of the <div> tag?

Answer: The <div> tag is a block-level element used to group other HTML elements and apply CSS styles or JavaScript functionality to them.


Question 8: How can I make a list in HTML?

Answer: Use <ul> for unordered lists, <ol> for ordered lists, and <li> for list items. Example: <ul><li>Item 1</li></ul>


Question 9: What is the <form> element used for?

Answer: The <form> element is used to collect user input and submit it to a server. It can contain input fields, buttons, and other interactive controls.
